# Wifi Communication - ButtonPresser
communicating between two microcontrollers via wifi access points

Microcontrollers:
- [Adafruit ESP32 Feather V2](https://www.adafruit.com/product/5400?srsltid=AfmBOorb39kMJ66T4G6jx-E4MQ4UTF1b23FAYdZPfULnOvC8R-Qn7FOE) - Access Point, Parent
- [Arduino UNO R4 Wifi](https://store-usa.arduino.cc/products/uno-r4-wifi?srsltid=AfmBOoo8Gmp6grwCyCvPOAqdj7CC2VMUsgU0qByGo3Ea6UkLQIJfED6C) - Reveiver, Child

## Setting up
- For setting up Adafruit's ESP32 Feather V2 Board, See [Dr. Sudhu's Tutorial](https://github.com/loopstick/ESP32_V2_Tutorial)
- [Code for using Wifi with ESP32 V2](https://github.com/romyaboudarham/ArduinoR4Wifi-ButtonPresser/blob/main/ESP32_V2-AccessPoint/ESP32_V2-AccessPoint.ino)
- [Code for using Wifi with UNO R4](https://github.com/romyaboudarham/ArduinoR4Wifi-ButtonPresser/blob/main/Arduino_UNO_R4-Receiver/Arduino_UNO_R4-Receiver.ino)
